Code and Pre formatting tags

If your code is of single line use code tag. If your code is multiline use pre tag

Using code tag

#include void main() { int a,b,c; printf(" Enter the value of a"); }

Using pre tag

                #include
                        void main()
                        {
                                int a,b,c;
                                printf(" Enter the value of a");

                        }

Pre tag will shows the exact content what we have written along with spaces

Meanwhile code tag ignores all the spaces and shows code in single line
